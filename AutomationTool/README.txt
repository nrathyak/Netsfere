#########################################################################################################
									NetSfere Automation Tool
##########################################################################################################

This is an automation tool developed to test NetSfere Secure Messaging Application.
This tool covers multiple platforms like Android, IOS, Web

-> The list of test cases this tool covers can be found at below link
https://docs.google.com/spreadsheets/d/1Sc7r2IRDDW9rp1c8wAoRY3afSWzbo0qgxImv92WZuuw/edit?usp=sharing

NOTE:Attachment test cases for Android will work only for Nexus 5X device.

-> To checkout this project : http://blrsvn.mp.ics.com/svn/netsfere/trunk/test/Automation

-> To run :
	End to end sanity :
		Right click on testng.xml file in root directory -> Run AS -> TestNG Suite
	Android sanity :
		Right click on androidSanity.xml file in root directory -> Run AS -> TestNG Suite

-> To configure : 
	General configurations :
		Edit config.properties file in root directory
	User account configurations :
